## Description ##

A WordPress plugin for storing and maintaining addresses.

## Installation ##

### Manual Installation ###

1. Upload the entire `/address-book` directory to the `/wp-content/plugins/` directory.
2. Activate Address Book through the 'Plugins' menu in WordPress.
